# ssh-reg

````
usage: ssh-reg [<flags>] <command> [<args> ...]

A ssh config management tool.

Flags:
  --help  Show help (also see --help-long and --help-man).

Commands:
  help [<command>...]
    Show help.

  list
    List all available hosts

  add [<flags>] <host> <hostname>
    Add host

  remove <host>
    Remove host

  update [<flags>] <host> [<hostname>]
    Update host
````
